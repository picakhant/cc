### 🧑‍🏫 **Teacher Upload Data Access Guide**

သင်၏ upload file များကို သက်ဆိုင်ရာ class ၏ folder အောက်ရှိ `teacher_uploads` folder အောက်တွင် ကြည့်ရှုနိုင်ပါသည်။

**For example:** ClassBridge/class_rooms/teacher_uploads/file.zip

> 💡 **Note:**
>
> - File များအပေါ်တွင် **CRUD operation (create, read, update, delete)** ပြုလုပ်နိုင်ပါသည်။
> - သို့သော် **directory (folder)** များအပေါ်တွင် CRUD operation မပြုလုပ်ပါနှင့်။
> - Folder structure ကို system မှ automatically စီမံသည့်အတွက် ဖိုင်အဆင့်တွင်သာ လုပ်ဆောင်ရန် အကြံပြုပါသည်။
