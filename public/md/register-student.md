### 📘 **Student Register လုပ်နည်း**

##### 🧾 **အဆင့် (၁) — Text File တစ်ခု Create ပြုလုပ်ပါ။**

ပထမဦးဆုံး `.txt` extension ပါသော text file တစ်ခု create ပြုလုပ်ပါ။

ဥပမာ - students.txt

---

##### ✍️ **အဆင့် (၂) — ကျောင်းသားအချက်အလက် ထည့်ပါ။**

ဖိုင်အတွင်းတွင် အောက်ပါ format အတိုင်း input ထည့်ပါ။  
ကျောင်းသားတစ်ယောက်ချင်းစီကို **comma ( ; )** ဖြင့် ခွဲရေးပါ။

**Format ➤** rowNumber: name: major: year

---

##### 🧩 **ဥပမာ**

002296: Mg Mg: CS: 4th year;
002298: Hla Hla: CS: 4th year;

---

##### 💡 **မှတ်ချက်**

- comma ( , ) မရှိပါက system မှတစ်ယောက်ချင်းစီကို ခွဲမရနိုင်ပါ။
- `:` (colon) သုံးရာတွင် space မပါအောင်ရေးပါ။
- ဖိုင်ကို save ပြုလုပ်ပြီးမှ upload ပြုလုပ်ပါ။
