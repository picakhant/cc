### 🏫 Class Management Guide

#### 📁 **အတန်းများ ဖန်တီးခြင်း**

မိမိသင်ကြားသည့် အတန်းများကို **Add New Class** button ကို click ပြီး **class folder** များ ဖန်တီးနိုင်ပါသည်။  
**ClassBridge** folder အောက်တွင် ဖန်တီးထားသော **class folder** များကို `class_rooms` folder အောက်တွင် ကြည့်ရှုနိုင်ပါသည်။  

class folder များအား **OS level** မှတဆင့်လည်း `class_rooms` folder အောက်တွင် **create / update / delete** ပြုလုပ်နိုင်ပါသည်။

> ⚠️ **Note:** `class_rooms` folder အောက်တွင်သာ ဖန်တီးပါ။

---

#### 🧑‍🎓 **ကျောင်းသားများ Register ပြုလုပ်ခြင်း**

ကျောင်းသားများအား အတန်းအလိုက် register ပြုလုပ်ရန် **Register Students** button အား click ၍ ဖန်တီးနိုင်ပါသည်။  
Register ပြုလုပ်ထားသော ကျောင်းသားများစာရင်းကို `class_rooms` folder ၏ သက်ဆိုင်ရာ class folder အတွင်းရှိ **list.json** file အတွင်းတွင် ကြည့်နိုင်ပါသည်။

**For instance:**
/class_rooms/yourclass/list.json


> ⚠️ **Note:** `list.json` file အား **OS level** မှ update, rename သို့မဟုတ် CRUD operation များ ပြုလုပ်ခြင်းကို ရှောင်ရှားပါ။  
> Web interface အတွင်းမှသာ အသုံးပြုပါ။
